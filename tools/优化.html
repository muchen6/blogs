# 优化网页性能

# 懒加载和预加载

# 懒加载

长网页中延迟加载图像，用户滚动到它们之前，可视区域外的图像不会加载。

# 优点
  • 提升用户的体验(减少等待时间)
  • 减少无效资源的加载(减少服务器的压力和流量，减小浏览器的负担)
  • 防止并发加载的资源过多会阻塞js的加载

# 预加载

将所有所需的资源提前请求加载到本地，在需要用到时直接从缓存取资源。

# 优点

在网页全部加载之前，对一些主要内容进行加载，以提供给用户更好的体验，减少等待的时间。

# 区别

懒加载是迟缓甚至不加载，预加载是提前加载。

懒加载对服务器前端有一定的缓解压力作用，预加载则会增加服务器前端压力。
